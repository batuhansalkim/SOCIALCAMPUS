## 🚀 KURULUM ADIMLARI

### 1. Temel Sistem Gereksinimleri:
- Node.js v22.16.0+
- npm 10.9.2+
- JDK 17 (Eclipse Adoptium önerilen)
- Android Studio
- Android SDK 35
- Android Build Tools 35.0.0

### 2. Expo CLI Kurulumu:
```bash
npm install -g @expo/cli
```

### 3. Proje Oluşturma:
```bash
npx create-expo-app@latest ProjeAdi --template
cd ProjeAdi
```

### 4. Gerekli Paketlerin Kurulumu:
```bash
# Temel Expo paketleri
npx expo install expo-router expo-location expo-image-picker
npx expo install expo-splash-screen expo-linear-gradient
npx expo install expo-haptics expo-font expo-constants

# React Native paketleri
npx expo install react-native-maps react-native-paper
npx expo install react-native-nfc-manager react-native-ble-manager
npx expo install react-native-gesture-handler react-native-reanimated
npx expo install react-native-safe-area-context react-native-screens

# Blockchain paketleri
npm install ethers @walletconnect/modal @pinata/sdk
```

### 5. Android Konfigürasyonu:
- android/build.gradle dosyasında versiyonları ayarla
- android/gradle.properties dosyasında özellikleri yapılandır
- android/app/build.gradle dosyasında app konfigürasyonunu düzenle

## 📱 ÇALIŞTIRMA YÖNTEMLERİ

### 1. Expo Go ile Çalıştırma:
```bash
npx expo start
# QR kodu Expo Go uygulamasıyla tarayın
```

### 2. Android Emulator ile Çalıştırma:
```bash
npx expo run:android
# Android Studio emulator otomatik başlar
```

### 3. Development Build:
```bash
npx expo install expo-dev-client
npx expo run:android --variant debug
```

### 4. Web ile Çalıştırma:
```bash
npx expo start --web
```
